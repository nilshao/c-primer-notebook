# 第一章

## 操作系统的概念

1. 负责管理协调硬件、软件等计算机资源的工作

2. 为上层的应用程序，用户提供简单易用的服务。

3. 操作系统是系统软件，不是硬件

定义：指控制和管理整个计算机系统的硬件和软件资源，并合理的组织调度计算机的工作和资源的分配，以提供给用户和其他软件方便的接口和环境。它是计算机系统中最基本的系统软件
